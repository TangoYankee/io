Projects

This website is running on an Amazon EC2 instance. It is served by NGINX. The security certificates are provided by Let's Encrypt and installed with CertBot. The home page is a static site saved directly to the server. Spark Website is written with jekyll and was the original website for the Travis AFB Innovation Cell. Safebrowse is a Slack Application written in Node. Fundamentals serves a blog of CS concepts, written in flask with markdown pages. Safe Browse, Spark, and fundamentals are each running on a separate docker container in the same EC2 instance.
